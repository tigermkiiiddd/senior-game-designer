# 功能可见性暗示示例

## UI/HUD 示例

- **按钮**：内陷阴影、圆角 → 暗示可按
- **滑块**：带手柄的水平轨道 → 暗示可拖动
- **切换**：开关外观 → 暗示可翻转/旋转

## 环境示例

- **带手柄的箱子**：暗示可打开
- **杠杆**：暗示可拉/推
- **压力板**：暗示可踩在上面

## 谜题进阶示例

1. 展示基本四边形箱子（低功能可见性）
2. 添加颜色高亮（中等功能可见性）
3. 添加阴影/深度（较高功能可见性）
4. 添加可见手柄（最高功能可见性 → 清晰互动）

## 触屏界面示例

**我的马游戏**：需要手指追踪的圆形指示器——通过显示精确手势路径最大化功能可见性。
