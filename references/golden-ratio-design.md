# 设计中的黄金比例（Φ）

## 何时使用
- 设计 UI 布局、游戏板或矩形元素
- 创建建筑比例或环境艺术
- 需要建立自然悦目或有吸引力的比例
- 想要通过违反比例来创造刻意的不适或张力

## 核心概念
黄金比例（Phi，φ = 1.61803398875）是一种在自然界中普遍存在的数学比例，自约公元前 450 年起就用于设计。

**公式：** (a + b) / a = a / b
- a = 较大的量
- b = 较小的量
- a + b = 整体

## 执行步骤

### 1. 定义你的比例
- 识别你正在处理的尺寸（宽度、高度、间距）
- 确定哪个尺寸代表较大（a）和较小（b）的量

### 2. 应用黄金比例获得正面体验
- 设置比例使 (a + b) / a = 1.618
- 对于矩形：短边 = a，长边 = a + b
- 使用 Fibonacci 序列（1, 1, 2, 3, 5, 8, 13, 21, 34...）作为渐进方法
- 应用于 UI 元素大小、布局网格或空间关系

### 3. 用于负面体验（张力）
- 故意违反黄金比例以创造下意识的不适
- 对不安稳的环境、恐怖游戏或特定叙事情绪有用
- 考虑偏离 1.618 如何影响玩家/用户感知

### 4. 通过测试验证
- 观察用户/玩家对比例的反应
- 根据反馈和预期情感反应调整

## 常见应用
- **UI 设计：** 按钮大小、布局网格、元素间距
- **建筑：** 建筑立面、房间比例、窗户放置
- **环境艺术：** 自然形态、结构、景观构图
- **游戏板：** 桌游布局、卡牌尺寸、游戏区域

## 关键考量
- 比例可以近似（1.6 通常就足够了）
- 上下文重要——文化和设计惯例可能覆盖数学完美
- 用作指导原则，而非严格规则
- 在美学旁边考虑可访问性和可用性
