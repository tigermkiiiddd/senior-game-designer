# 游戏能力与谜题设计示例

## 记忆游戏示例
依赖于试错、记忆和模式识别的挑战：
- **FPS 关卡布局**：学习地图几何和生成点。
- **平台跳跃模式**：记忆跳跃的时机和序列。

## 技能游戏示例
需要身体或心智能力的挑战：
- **台球**：计算角度和理解物理。
- **RPG 策略**：规划构建和资源管理。

## 辅助机制示例
减少技能型游戏中沮丧的具体实现：
- **路易吉引导（超级马里奥 Wii）**：AI 角色控制以完成困难部分。
- **教程传送（鬼屋大冒险）**：将玩家飞回第一个世界重新学习基础。
- **视觉路径**：闪烁箭头或高亮交互对象。
- **环境利用**：用敌人身体或抛射物到达更高平台。
